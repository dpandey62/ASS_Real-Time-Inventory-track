# 🚀 Real-Time Inventory Tracker

A small, well-structured inventory management system built with Flask (Backend), React (Frontend), and a Relational Database.
Designed to demonstrate clarity, correctness, and maintainability — aligned with Better Software’s evaluation principles.

# 📌 Overview

This system allows users to:

Add, update, and delete inventory items

Track stock levels in real-time

Prevent invalid states (e.g., negative stock)

View current inventory in a clean UI

The focus is not on feature quantity, but on system design quality.

# 🧱 Tech Stack

# Layer         #	Technology

- Frontend	:         React.js

- Backend	 :          Python + Flask (REST API)

- Database	:         MYSQL

- API Calls	Axios

-Styling	Tailwind CSS (optional)

## ⚡ Setup Instructions

1.  Clone Repo:
   ```
   git clone https://github.com/dpandey62/ASS_Real-Time-Inventory-track.git
   cd ASS_Real-Time-Inventory-track
   ```

2. Backend Setup (Flask)
   
   ```
   cd backend
   
   pip install -r requirements.txt
   
   python app.py
   
   ```

3. Frontend Setup (React)
   
   ```
   cd frontend
   npm install
   npm start
   ```
   
4. Finally, start the Frontend of the Application:
   ```
   npm start
   ```

5. Open your browser and visit `http://localhost:3000` to access the captivating world of TravelDiary.
